##  输出

```js
    var a = 1;
    document.write(a);   // 窗口内文档
    alter(a);           //  窗口弹出条框
    console.log(a);     //  控制台输出结果
```

## 数据类型 数字

    字面量与进制
```js
    十进制     var a = 10; 

    八进制     var b = 012; // 10

    十六进制   var c =0x23; //  35
```
