# 数组

[数组(MDN文档)](https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Global_Objects/Array)

```js
var a = [1,2,3];  // 定义数组

// a.length    数组的长度

for（var i = 0; i < a.length; i++){
    console.log(a[i]);
}       // 遍历数组

// a[i] 都是元素，是一个空间，是变量，其中的值可以修改

```
