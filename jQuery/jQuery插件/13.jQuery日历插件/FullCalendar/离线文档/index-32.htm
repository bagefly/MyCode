Available Views

FullCalendar has a number of different "views", or ways of displaying days and events.

The following 5 views are all built in to FullCalendar:

  • month - see example
  • basicWeek - see example
  • basicDay - see example
  • agendaWeek - see example
  • agendaDay - see example

You can define header buttons to allow the user to switch between them. You can set the initial view of the calendar with the defaultView option.
